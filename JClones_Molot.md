# JClones_Molot
JClones_Molot is a clone of colorful, character compressor from the late 2000s / early 2010s

## Main Parameters

**Input:** Controls how hard the signal hits the compressor.

**SC Filter Freq.:** Reduces low-frequency influence on compression.

**SC Filter:** Enables or disables the sidechain filter.

**Threshold:** Sets the level where compression begins.

**Ratio:** Determines how strongly the signal is compressed after crossing threshold.

**Knee:** Controls how gradually compression starts. Soft knee (smaller values): Smoother, more musical onset. Hard knee (larger values): More aggressive, punchy. In Molot, Knee has a very audible effect - more than in many compressors.

**Attack:** Controls how fast compression engages.

**Attack Mode:** Switches between two detection behaviors.

**Alpha:** Smoother, more “musical”, less aggressive.

**Sigma:** More reactive, punchier, stronger transient shaping. 

**Release:** Controls how fast compression recovers.

**Sat:** Enables or disables the saturation.

**Sat Drive:** Increasing the level introduces harmonics.

**Makeup:** Adjusts the final level after compression.

**Dry Mix:** Blend compressed and original signal.

**Mode:** Sets stereo processing mode.

**Bypass:** Switches the processing off.

**Sidechain:** Switches between internal and external sidechain.

Note: JClones version of Molot doesn't include limiter, EQ and oversampling stages present in the original plugin.
